# 🏨 Hotel Analytics Dashboard — Tableau

An interactive Tableau dashboard providing a comprehensive overview 
of hotel property performance, built to surface actionable insights 
for hospitality business decision-making.

## 📊 Dashboard Highlights

- Hotel occupancy and booking trends
- Revenue performance across properties
- Guest demographics and stay patterns
- Key hospitality KPIs at a glance

## 🔗 Live Dashboard

👉 [View on Tableau Public](https://public.tableau.com/views/HospitalityOverview/Dashboard1?:language=en-US&:display_count=n&:origin=viz_share_link)

## 🗂️ Repository Structure

| File | Description |
|------|-------------|
| `Hospitality Overview.twbx` | Tableau packaged workbook |
| `hospitality_data.csv` | Raw dataset used for analysis |

## 🛠️ Tools Used

- **Tableau Desktop** — Dashboard design and visualizations
- **CSV** — Data source

## 🚀 How to Use

1. Download `Hospitality Overview.twbx`
2. Open with [Tableau Desktop](https://www.tableau.com/products/desktop)
3. Or view the live version directly on Tableau Public via the link above
